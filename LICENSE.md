## 🔧 Common Issues and Fixes

**❌ macOS Gatekeeper blocks Hydrogen from opening**
→ Go to *System Settings → Privacy & Security* and click **Open Anyway**
  next to the Hydrogen entry. This is a one-time step required for all
  unsigned Mac applications.

**❌ Key verification never completes**
→ Clear your browser's cache and cookies, then restart the key process
  from a fresh browser tab. Avoid using VPNs or ad-blockers during
  verification as they can block checkpoint detection.

**❌ Injection fails after a Roblox update**
→ Check the official **hydrogen mac discord** server for bypass status.
  Update Hydrogen to the latest version, fully restart both Roblox and
  Hydrogen, and retry after the game has fully loaded.

**❌ Script executes but nothing happens in-game**
→ The script may be outdated for the current game version. Browse the
  built-in Script Hub for an actively maintained alternative.

**❌ Hydrogen crashes on Apple Silicon Mac after 30–40 minutes**
→ This was resolved in v2.706. Update to the latest version. If the
  issue persists, close background applications to free RAM before
  starting long sessions.

**❌ Android APK installs but crashes on launch**
→ Ensure your device runs Android 7.0 or later. Fully uninstall any
  previous Hydrogen version before installing the new APK.

**❌ Script Hub fails to load or shows blank**
→ Check your internet connection. The Script Hub requires an active
  connection to fetch content. If connection is stable, restart the
  Hydrogen app and try again.
